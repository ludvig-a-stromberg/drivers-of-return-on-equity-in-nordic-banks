# Drivers of Return on Equity in Nordic Banks

This project analyzes the drivers of return on equity (ROE) among Nordic banks using a DuPont-based profitability decomposition and regression analysis.

Using financial statement data from SEB, Nordea, and Handelsbanken (2014–2025), the study examines how differences in revenue structure, cost intensity, credit losses, and capital structure explain variations in bank profitability.

The analysis finds that fee income intensity is the strongest differentiator of ROE across banks, while cost efficiency, credit losses, and capital ratios play smaller but significant roles.

The full analysis and methodology are documented in the R Markdown notebook and accompanying report.
